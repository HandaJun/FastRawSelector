# FastRawSelector 배포 · 포터블 안내

> D-1 · 2026-07-16  
> 대상: .NET Framework 4.6.2 / Windows / 단일 exe (Costura.Fody)

---

## 1. 런타임 전제

| 항목 | 내용 |
|---|---|
| OS | Windows 7 SP1 이상 (WPF) |
| 런타임 | **.NET Framework 4.6.2** 이상 (미설치 PC는 [공식 런타임](https://dotnet.microsoft.com/download/dotnet-framework/net462) 필요) |
| 아키텍처 | AnyCPU, Prefer32Bit=false → 64bit OS 에서는 64bit 프로세스 |
| 권한 | 일반 사용자 OK. 탐색기 연동은 **HKCU / SendTo** 만 사용 (관리자 불필요) |

---

## 2. 데이터 위치 (AppData)

설치 경로와 분리되어 있습니다.

```
%AppData%\Roaming\FastRawSelector\
  FastRawSelectorSetting.yaml   # 전역 설정
  logs\                         # 로그
  native\                       # libraw / exiv2 추출본
  thumbs\                       # 그리드 썸네일 디스크 캐시
```

사진 폴더별 선택 상태:

```
<사진폴더>\FastRawSelector.yaml
```

**포터블 zip**으로 USB에 넣어도 설정·로그는 위 AppData에 남습니다.  
완전 포터블(설정까지 USB)은 현재 미지원입니다.

---

## 3. 포터블 zip 배포 절차

### 빌드

```bat
nuget restore src\FastRawSelector.sln
msbuild src\FastRawSelector.sln /p:Configuration=Release /p:Platform="Any CPU"
```

출력:

```
src\FastRawSelector\bin\Release\FastRawSelector.exe
```

Costura가 관리 어셈블리를 exe에 포함합니다.  
네이티브 DLL은 첫 실행 시 AppData\native 으로 추출됩니다.

### zip 구성 (최소)

```
FastRawSelector/
  FastRawSelector.exe
  (선택) README.txt  — 본 문서 요약
```

`packages\`, `pdb`, `xml` 문서는 배포 불필요.

### 사용자 안내 문구 예시

1. zip 압축 해제  
2. .NET Framework 4.6.2 이상 설치 여부 확인  
3. `FastRawSelector.exe` 실행  
4. RAW 폴더를 드래그하거나 **파일 → 열기**  
5. (선택) **설정**에서 보내기 / 연결 프로그램 등록

## 4. “설치” 형태 (권장 수동)

별도 MSI/인스톨러는 아직 없습니다. 아래 중 하나면 충분합니다.

| 방식 | 방법 |
|---|---|
| 사용자 폴더 | `%LocalAppData%\Programs\FastRawSelector\` 에 exe 복사 후 바로가기 |
| 프로그램 파일 | `%ProgramFiles%\FastRawSelector\` — **exe 디렉터리 쓰기 불필요** (네이티브는 AppData) |
| 시작 메뉴 | 바로가기로 등록 |

Program Files에 넣어도 AppData 추출 방식이라 **설치 폴더 쓰기 권한은 필수가 아닙니다.**

---

## 5. 탐색기 연동 (D-2, 설정 UI)

설정 창 → **탐색기 연동 (현재 사용자)**:

| 옵션 | 동작 |
|---|---|
| 보내기 메뉴 등록 | `%AppData%\Microsoft\Windows\SendTo\FastRawSelector.lnk` |
| 연결 프로그램 목록 | `HKCU\Software\Classes\Applications\FastRawSelector.exe` + RAW `SupportedTypes` |

- 기본 프로그램 강제 지정 없음 (사용자가 연결 프로그램에서 선택)  
- 체크 해제 시 등록 제거  
- 관리자 권한 불필요  

---

## 6. 문제 해결

| 증상 | 확인 |
|---|---|
| 실행 안 됨 | .NET 4.6.2 설치, 로그 폴더 생성 여부 |
| RAW 안 열림 | `logs\` ERROR, native DLL 추출 여부 (`native\`) |
| 한글 경로 | libraw open_wfile 사용 중 — 최신 빌드 확인 |
| SendTo 안 보임 | 탐색기 재시작, 설정에서 재등록 |

## 8. 버전 · GitHub Release 게시 (개발자)

### 버전 규칙

| 항목 | 규칙 |
|---|---|
| Assembly | `src/FastRawSelector/Properties/AssemblyInfo.cs` — `AssemblyVersion` / `AssemblyFileVersion` |
| 형식 | `Major.Minor.Patch.0` (예: `1.0.0.0`) |
| Git 태그 | `vMajor.Minor.Patch` (예: `v1.0.0`) |
| 창 제목 | `Common.GetVersion(..., " v", 3)` → ` v1.0.0` |

4) zip 구성:

       FastRawSelector.exe
       README.txt   (선택 — 아래 문구)
5) git commit / tag vX.Y.Z / push --tags
6) GitHub Releases → Draft a release

   - 본문: 변경 요약 (한국어 OK)
